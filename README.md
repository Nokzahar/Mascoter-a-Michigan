# Proyecto Mascotería Michigan

## Descripción
Este proyecto busca aprender a implementar el uso de Git y Github simulando el trabajo en equipo de varias personas en un solo código.

## Funcionalidades Implementadas
* **Estructura base de la pagina**: Cree los archivos principales de la página como el index y el README.
* **Gestión de Repositorio**: Apliqué las configuraciones básicas de Git, cree también un repositorio local y lo vinculé con un repositorio remoto en GitHub.
* **Barra de navegación (Navbar)**: Implementé un encabezado básico en el archivo HTML.
* **Pie de página (Footer)**: Implementé un pie de página para simular el trabajo en equipo con Pull Request.
* **Uso de Markdown**: Aplicando el uso de Markdown para describir este proyecto en el archivo README.

## Ramas de Git utilizadas
* `main`: Rama principal con el código estable.
* `feature/navbar`: Creación de la barra de navegación
* `feature/footer`: Implementación del pie de página mediante Pull Request
